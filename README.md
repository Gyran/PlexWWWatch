PlexWWWatch
===========
A web front-end for [Plex Watch](https://github.com/ljunkie/plexWatch)

Features
---------------------
Early development, only history at the moment

Prerequisites
-----------------
* Sqlite3
* Server that can serve PHP files

Installation
-------------------
* Clone repo to desired location
* Visit http://host/PlexWWWatch
* Visit settings page
* Profit...
