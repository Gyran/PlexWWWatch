If you need authentication for local servers or if PlexWWWatch is not running on the same local network as your Plex Media Server a Plex token is required for authentication.
To acquire a token, sign in to plex.tv above and click "FETCH TOKEN" (your username or password are never stored) and wait until the token apear in the token field. Then save the settings.
